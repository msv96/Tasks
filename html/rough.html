typeof in JS

  1. typeof(1) is number
  2. typeof(1.1) is number
  3. typeof('1.1') is string
  4. typeof(true) is boolean
  5. typeof(NaN) is number
  6. typeof(undefined) is undefined
  7. typeof(null) is object
  8. typeof([]) is object
  9. typeof({}) is object

Differences between 'div', 'section' and 'article' tags

Div Tag

Div tag is used for non-relative and used for styling in CSS

Section Tag

Section tag is used when the contents are relative and not self-contained

Article Tag

Article tag is used when the contents are relative and self-contained and should have a heading
